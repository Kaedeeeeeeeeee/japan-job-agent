# Japan Job Agent specification bundle v0.1

This bundle contains:

- `japan-job-agent-development-spec-v0.1.md`: implementation-ready development specification.
- `schema-v0.1.sql`: executable PostgreSQL baseline schema.
- `types-v0.1.ts`: TypeScript domain and connector contracts.

The Markdown specification is the source of truth for scope, architecture, decisions, workflows and acceptance criteria. The SQL and TypeScript files are implementation companions and should be changed through reviewed migrations or architecture decision records.
