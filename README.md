# Japan Job Agent

日本の企業公式サイトと ATS を検証し、応募可能な求人を証拠付きで推薦する個人向け Job Agent です。

`main` の初回コミットは、実装前に確定した v0.1 仕様の読み取り専用スナップショットです。v0.2 以降の変更は Pull Request を通して追加します。
